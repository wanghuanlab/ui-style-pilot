# 01 设计 Token

设计 Token 是项目 UI 的唯一视觉来源。新增页面和组件应从 `src/root.scss`、`docs/frontend-standards/00-design-source.md` 继承颜色、字体、间距、圆角和层级，不允许在业务页面自由发明新视觉体系。

## 色彩

| 语义 | 值 | 用途 |
|---|---:|---|
| 主色 | `#005BAC` | 主按钮、导航选中、表头强调、关键数据 |
| 主色 Hover | `#004892` | 主按钮 hover、链接 hover、菜单 hover |
| 主色 Active | `#002F61` | 按下态、强选中态 |
| 次级交互色 | `#55C4F1` | 次级高亮、辅助图表色、弱提示 |
| 禁用蓝 | `#B2CDE6` | 禁用按钮、不可交互状态 |
| 浅蓝底 | `#EAF1F9` | 选中背景、筛选区、轻提示 |
| 页面浅灰底 | `#F2F2F2` | 后台页面画布 |
| 白底 | `#FFFFFF` | 卡片、表格、弹窗、抽屉 |
| 成功 | `#00A85A` | 正常、通过、完成、启用 |
| 警示 | `#F6AD55` | 待处理、审批中、警告 |
| 错误 | `#DC3C3A` | 驳回、失败、危险、删除 |

文本色使用黑色透明度语义：

| 语义 | 建议值 | 用途 |
|---|---:|---|
| 主文本 | `rgba(0, 0, 0, 0.85)` | 页面标题、表格正文、重要字段 |
| 次文本 | `rgba(0, 0, 0, 0.65)` | 辅助信息、表单说明 |
| 弱文本 | `rgba(0, 0, 0, 0.45)` | 占位文字、弱提示 |
| 禁用文本 | `rgba(0, 0, 0, 0.35)` | 禁用状态 |
| 默认边框 | `rgba(0, 0, 0, 0.15)` | 输入框、卡片、表格分隔 |

## 字体

字体栈：

```css
'Source Han Sans SC', 'Noto Sans SC', 'Microsoft YaHei', 'PingFang SC', 'Segoe UI', Arial, sans-serif
```

| 层级 | 字号 | 字重 | 行高 | 用途 |
|---|---:|---:|---:|---|
| 页面标题 | 18px | 600 | 27px | 页面主标题 |
| 区块标题 | 16px | 500/600 | 24px | 卡片标题、表格标题 |
| 正文 | 14px | 400 | 21px | 表单、普通文本 |
| 紧凑文本 | 12px | 400/500 | 18px | 表格、筛选、分页、图表 |
| 辅助标签 | 12/13px | 400 | 18/20px | 图例、单位、说明 |

新页面不得随意使用 `11px`、`15px`、`17px`、`20px+` 等非规范字号。确有必要时，应先沉淀为 Token 或公共组件规则。

## 间距

项目使用 4 倍数间距：

| Token | 值 | 用途 |
|---|---:|---|
| `space-1` | 4px | 图标与文字、紧凑元素内部 |
| `space-2` | 8px | 表单项内部、按钮组、表格操作 |
| `space-4` | 16px | 组件之间、卡片内边距 |
| `space-6` | 24px | 区块之间、页面主要间距 |
| `space-12` | 48px | 大区块间距、特殊空状态 |

业务页面禁止出现大量非体系间距，例如 `margin: 13px`、`padding: 17px`。如果是为对齐组件库内部结构，可在公共组件内处理。

## 圆角与边框

| 类型 | 值 | 用途 |
|---|---:|---|
| 控件圆角 | 4px | 按钮、输入框、选择器、分页 |
| 容器圆角 | 4px/6px | 卡片、筛选区、表格外框 |
| 分页圆角 | 3px/4px | 页码按钮 |
| 边框 | 1px solid `border-default` | 控件、卡片、表格 |

不要在业务页面随意使用大圆角，如 `12px`、`16px`、`24px`，除非公共组件明确规定。

## 阴影

后台系统以低干扰为原则：

- 常规卡片：优先边框，不使用重阴影。
- 弹层、下拉、Popconfirm：允许轻阴影。
- Header 和侧边栏：允许弱阴影强化层级。

避免大面积高对比投影、渐变光晕、装饰性背景。
