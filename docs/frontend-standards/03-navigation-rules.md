# 03 导航规则

导航承载系统信息架构，风格应稳定、低噪音、易扫描。不要在不同模块中改变导航尺寸、选中态或展开方式。

## 顶部导航

| 项 | 规范 |
|---|---|
| 高度 | 60px |
| 背景 | 主色深蓝或品牌渐变，但同一系统保持一致 |
| Logo | 按视觉规范使用，不拉伸 |
| 工具图标 | 20px 左右 |
| 头像 | 24px/32px，按全局 Header 统一 |
| 标题 | 16px/18px，白色，字重 600 |

顶部导航只放全局能力：

- 系统标识
- 全局搜索
- 通知
- 用户菜单
- 原型/API 模式等开发态能力

不要在顶部导航放页面级业务按钮。

## 侧边导航

| 项 | 规范 |
|---|---|
| 展开宽度 | 240px |
| 收起宽度 | 64px |
| 背景 | 白色或极浅色 |
| 一级菜单字号 | 14px |
| 二级/三级菜单字号 | 14px 或 13px |
| 图标 | 16px/18px |
| 菜单项高度 | 40px 左右 |
| 选中态 | 主色背景或主色文字，必须明显 |

当前历史实现如宽度为 252px，可在后续布局统一时逐步收敛到 240px。

## 菜单层级

- 菜单层级原则上不超过三级。
- 当前模块应展开到可见当前页面。
- 当前页面菜单项必须有明确选中态。
- 折叠菜单时，图标必须保留可识别性。

## 菜单搜索

菜单项多时允许顶部放搜索框：

- 搜索框高度 28px/32px。
- 占位文字使用弱文本色。
- 搜索结果应保留层级上下文。

## 面包屑

面包屑位于内容区顶部，格式：

```text
首页 / 一级模块 / 二级模块 / 当前页面
```

规则：

- 字号 14px 或 12px。
- 当前页面可以弱化为普通文本。
- 不要让面包屑和页面标题重复造成视觉拥堵。
- 1280 宽下长面包屑可省略中间层级，但必须保留当前页面。
