# 视觉规范到后台系统的设计映射

本文档基于上游视觉规范提炼，用于指导后台管理系统的 Token、组件、布局与图表落地。落地到你的项目时，请替换为团队自己的品牌色与设计资产。

## 设计 Token

| 规范内容 | 后台系统映射方式 | 实现建议 |
|---|---|---|
| 主题色：`#005BAC` 品牌主色 | 主按钮、选中态、导航高亮、关键数据强调 | 定义为 `color-primary`，禁止业务模块随意替换 |
| Hover：`#004892` | 主按钮 Hover、链接 Hover、选中控件 Hover | 定义为 `color-primary-hover` |
| Active：`#002F61` | 按下态、当前激活态、强选中态 | 定义为 `color-primary-active` |
| 次级交互：`#55C4F1` | 同层级次要高亮、辅助交互色 | 用于次按钮、辅助图表色、弱提示 |
| 禁用/不可交互：`#B2CDE6` | 禁用按钮、未满足条件控件 | 统一 disabled 色，不单独配置灰蓝变体 |
| 浅蓝底：`#EAF1F9` | 选中背景、轻量提示底、筛选区底色 | 适合表格筛选、标签选中、菜单 Hover 背景 |
| 浅灰底：`#F2F2F2` | 页面灰底、区块弱分隔 | 用作后台画布背景 |
| 白色：`#FFFFFF` | 卡片、弹窗、表格、内容容器底色 | 内容承载区域优先使用白底 |
| 成功：`#00A85A` | 成功提示、健康状态、正常运行 | 定义为 `color-success` |
| 警示：`#F6AD55` | 警告、备注、待处理状态 | 定义为 `color-warning` |
| 错误：`#DC3C3A` | 错误、危险、拒绝、校验失败 | 定义为 `color-error` |
| 中性色：黑色透明度 85%、65%、45%、35% | 标题、正文、辅助文字、图标层级 | 建议转为语义 Token：`text-primary`、`text-regular`、`text-secondary`、`text-disabled` |
| 边框：黑色 15% | 输入框、按钮、分页、卡片边界 | 定义为 `border-default` |
| 背景灰：黑色 5% | 页面底、Hover 弱背景 | 定义为 `bg-muted` |
| 字体：思源黑体优先 | 中文 UI 主字体 | 字体栈：`Source Han Sans SC, PingFang SC, Microsoft YaHei, sans-serif` |
| 字重：Regular 400、Medium 500、Bold 600 | 标题、按钮、正文层级 | 后台系统建议标题 500/600，正文 400 |
| 标题 18px，小标题 16px | 页面标题、模块标题、卡片标题 | 页面标题 18px，区块标题 16px |
| 正文舒适型 14px，紧凑型 12px | 表单、表格、筛选、密集信息区 | 默认 14px，表格紧凑模式 12px |
| 图表/辅助文字 12px、13px | 坐标轴、图例、单位、说明 | 图表默认 12px，重要标签 13px |
| 行高 = 字号 x 1.5 | 全局文本行高 | 14px 对应 21px，12px 对应 18px |
| 间距：4、8、16、24、48px | 全局 spacing scale | 定义为 `space-1/2/4/6/12` 或直接按 px Token |

## 组件规范

| 规范内容 | 后台系统映射方式 | 实现建议 |
|---|---|---|
| 按钮类型：主要、次要、默认、文本、幽灵 | 后台操作层级体系 | 每页原则上只保留一个主按钮，其他操作降级 |
| 按钮圆角 4px | 全局控件圆角 | 定义 `radius-control: 4px` |
| 按钮尺寸：舒适型 38px，紧凑型 28px | 表单页/首页用 L，表格行内/弹层用 S | 组件库提供 `size="large"` 与 `size="small"` |
| 按钮文案“动词+名词” | 操作语义清晰 | 如“新建任务”“导出报表”“取消订单” |
| 输入框类型：全局输入、搜索框、多行输入 | 表单、筛选、搜索、备注 | 统一状态：默认、输入、完成、禁用、错误 |
| 输入框高度：38px / 28px | 普通表单 / 紧凑筛选 | 与按钮高度保持一致 |
| 输入框错误态：边框 `#DC3C3A`，提示同色 | 表单校验 | 错误信息紧贴字段下方，不只用颜色提示 |
| 下拉菜单：基础、多选、多级 | 枚举选择、批量筛选、层级选择 | 选中背景用主色 10%，Hover 用 Hover 色 10% |
| 单选框/复选框 | 表单选择、筛选条件 | 默认边框黑 25%，选中背景主色 |
| 表格筛选区：单行、双行、分割筛选、分离筛选 | 数据列表页筛选布局 | 简单筛选单行；筛选多或批量操作多时使用双行/分离筛选 |
| 表格标题优先左上 | 明确当前数据对象 | 有标题时主按钮右对齐；无标题且操作少时按钮可左对齐 |
| 表格次要操作使用图标按钮 | 密度切换、列设置、全屏 | 放在表格工具栏右侧，降低视觉权重 |
| 表格尺寸：主表行高 = 文字 x 3，副表 = 文字 x 2 | 核心表格 / 次级表格 | 14px 主表行高约 42px；12px 紧凑表约 24-36px |
| 分页：复杂、基础、简洁 | 大数据列表、少量页面、轻量列表 | 大多数后台列表使用复杂分页 |
| 分页圆角 3px，选中色主色 | 页码控件 | 与按钮圆角略区分，保持轻量 |
| 顶部导航高度 60px | 全局 Header | Logo 132x34，工具图标 20px，头像 24px |
| 侧边导航宽度 240px，收起 64px | 后台主导航 | 业务层级深时用一级菜单+右侧平铺模块；层级浅时用树状导航 |
| 标签页：导航、基础、卡片、胶囊、文本 | 页面级/区块级内容切换 | 支持三级嵌套，但避免超过三级 |
| 标签页高度：基础/卡片 38px，导航/胶囊/文本 32px | 区分层级密度 | 页面级用 38px，轻量切换用 32px |
| 抽屉：右/左/上/下，多层抽屉 | 快速编辑、详情、辅助任务 | 同一业务内抽屉方向保持统一 |
| 抽屉尺寸：480/640/800/900px | S/M/L/XL 侧边任务面板 | 简单表单 480，复杂详情 640+ |
| 对话框：确认、消息提示 | 中断式确认或轻量录入 | 谨慎使用，频繁操作优先抽屉或页面内编辑 |
| 气泡确认窗 12 方位 | 局部提示、二次确认 | 删除、危险操作可用气泡确认 |
| 标签：通用、可关闭、可选择、彩色标签 | 状态、分类、筛选条件 | 状态标签用功能色，分类标签用彩色标签组 |

## 布局规则

| 规范内容 | 后台系统映射方式 | 实现建议 |
|---|---|---|
| 设计基准：1920x1080 | 大屏后台主设计稿尺寸 | 以 1920 宽为设计基准，同时做响应式适配 |
| 页面结构：上下结构 | 顶部导航 + 内容区 | 适合首页、驾驶舱、横向导航系统 |
| 页面结构：上左右结构 | 顶部导航 + 左筛选/导航 + 右内容 | 适合管理后台、列表详情、复杂筛选场景 |
| 左侧区域固定，右侧自适应 | 筛选栏、侧边菜单、树形目录 | 左侧 240px 或业务定宽，右侧 `flex: 1` |
| 12 等分栅格 | 信息分组少、单块信息体量大 | 用于概览卡片、表单分栏、常规后台页 |
| 24 等分栅格 | 信息量大、分组多、密集业务页面 | 用于复杂看板、多指标卡、多列筛选 |
| 水槽：8、16、24、32、40px | 栅格列间距 | 常规后台推荐 16/24；密集表格页可用 8/16 |
| 卡片、图表边缘对齐栅格线 | 保持页面秩序感 | 不要把卡片或图表放在水槽中 |
| 间距体系：4/8/16/24/48px | 控件、区块、页面级间距 | 4 用于图标文字，8 用于控件内，16 用于组件间，24 用于区块间，48 用于页面大分隔 |
| 页面背景浅灰，内容模块白底 | 后台管理系统基本层次 | 页面 `#F2F2F2`，卡片/表格/弹窗 `#FFFFFF` |
| 操作权重排序 | 标题/切换 > 筛选 > 主操作 > 次要工具 | 工具栏从左到右、从上到下体现信息优先级 |
| 表格复杂筛选可拆出独立筛选区 | 高复杂度查询页面 | 分割线或独立筛选容器承载，查询按钮左对齐 |

## 图表风格

| 规范内容 | 后台系统映射方式 | 实现建议 |
|---|---|---|
| 主题色用于图表配色基础 | 主系列、关键指标、高亮数据 | 第一主色使用 `#005BAC` |
| 次级交互色 `#55C4F1` | 第二系列、辅助趋势线 | 与主色形成蓝色系层级，但避免整图单色 |
| 功能色成功/警示/错误 | 状态类图表、阈值线、异常点 | 正常 `#00A85A`，预警 `#F6AD55`，异常 `#DC3C3A` |
| 彩色标签色组 | 多分类图表扩展色 | 可补充 `#0E6EB8`、`#0E95B8`、`#699B07`、`#E29500`、`#F1681E` |
| 图表文字 12px/13px | 坐标轴、图例、数据标签 | 轴标签 12px，图表标题/关键数值辅助 13px |
| 文本色 65%、45%、35% | 图例、轴线说明、辅助说明 | 图例用 65%，坐标轴/单位用 45%，弱提示用 35% |
| 图表容器白底 | 卡片式图表模块 | 与表格、卡片保持同一内容底色 |
| 图表边缘对齐栅格线 | 图表布局一致性 | 图表卡片宽度按 12/24 栅格分配 |
| 浅蓝底 `#EAF1F9` | 选中区域、Tooltip 背景、轻量高亮 | 用于 hover 区域或选中数据块 |
| 边框黑 15%、背景黑 5% | 网格线、分隔线、弱背景 | 网格线尽量轻，避免抢占数据视觉权重 |
| 行高 1.5 | 图表说明、Tooltip 多行文本 | Tooltip 内文本按 12/14px + 1.5 行高 |
| 后台长时间操作场景 | 图表应稳健、清晰、低刺激 | 避免高饱和大面积填充；关键异常才用强红/橙 |

## 落地关系

| 规范资产 | DEMO 文件 | 说明 |
|---|---|---|
| Color Tokens、Typography Tokens、Layout Tokens | `src/styles/tokens.css` | 集中定义平台级 CSS 变量 |
| Element Plus 组件覆盖 | `src/styles/app.css` | 按规范覆盖按钮、输入框、表格、分页、弹窗、标签等默认样式 |
| ECharts 主题映射 | `src/App.vue` | 图表色板、字体、网格线、Tooltip 与规范 Token 保持一致 |
| Mock 数据与业务状态 | `src/data/mock.js` | 统一维护项目、风险、指标、用户等演示数据 |
