# 11 图表规则

本文件用于约束 ECharts 和其他图表在后台系统中的视觉风格。图表应稳健、清晰、低刺激，优先服务业务判断，不追求装饰性效果。

## 色彩

| 用途 | 颜色 |
|---|---:|
| 主系列、关键指标、高亮数据 | `#005BAC` |
| 第二系列、辅助趋势线 | `#55C4F1` |
| 正常、健康、通过 | `#00A85A` |
| 预警、待处理 | `#F6AD55` |
| 异常、错误、超限 | `#DC3C3A` |

多分类图表可补充：

```text
#0E6EB8
#0E95B8
#699B07
#E29500
#F1681E
```

避免整图只使用同一蓝色的深浅变化。蓝色可作为主基调，但分类、状态、异常必须有清晰区分。

## 字体与文本

| 元素 | 字号 | 颜色 |
|---|---:|---|
| 坐标轴 | 12px | `rgba(0, 0, 0, 0.45)` |
| 图例 | 12px | `rgba(0, 0, 0, 0.65)` |
| 单位、说明 | 12px | `rgba(0, 0, 0, 0.45)` |
| 关键数据标签 | 13px | `rgba(0, 0, 0, 0.85)` |
| Tooltip | 12px/14px | 按内容层级使用 |

行高保持 `1.5`。不要在图表中使用过小文字，尤其是坐标轴和 Tooltip。

## 容器

- 图表容器使用白底。
- 图表卡片边缘应与表格、筛选区、指标卡对齐。
- 图表卡片优先使用边框，不使用重阴影。
- 网格线和分隔线应轻，不抢数据视觉权重。
- Hover 或选中区域可使用 `#EAF1F9` 作为浅蓝底。

## 布局

- 概览页使用 12/24 栅格。
- 图表宽度按栅格分配，不随意写特殊宽度。
- 图表和指标卡之间使用 16px 或 24px 间距。
- 图表标题使用 16px 区块标题。
- 图表说明文字使用 12px/13px。

## 状态表达

状态类图表必须使用功能色：

- 正常：成功色。
- 预警：警示色。
- 异常：错误色。

不要只用颜色表达状态。必要时同时使用文字、图标、阈值线或标签。

## Tooltip

Tooltip 应包含：

- 指标名称。
- 数值和单位。
- 时间或分类。
- 状态说明，若有异常。

Tooltip 背景应清晰可读，不使用高透明度导致文字对比不足。

## 禁止项

- 禁止高饱和大面积填充。
- 禁止装饰性 3D 图表。
- 禁止无意义渐变。
- 禁止多个分类全部使用相近蓝色。
- 禁止图例、坐标轴、单位文字过小。
- 禁止图表容器与页面栅格错位。

## 审查点

- 是否使用项目主色和功能色。
- 是否避免一图单色。
- 是否有清晰标题和单位。
- 是否有合理 Tooltip。
- 是否图表容器与页面布局对齐。
- 是否在异常或预警场景中使用功能色和文字共同表达。
